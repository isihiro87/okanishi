# 岡崎西高等学校 同窓会ホームページ 設計書

## 概要

岡崎西高校同窓会の公式ホームページ。
卒業生が「懐かしさ」と「つながり」を感じられる場を、
複数ページ構成で実現する。

技術: HTML + CSS + JavaScript（何回生計算機のみJSを使用）
デザイン準拠: `.claude/skills/alumni-homepage-design.md`

---

## サイトマップ

```
index.html .................. トップページ（ハブ）
├── about.html .............. 同窓会について
├── news.html ............... お知らせ一覧
│   └── news-detail.html .... お知らせ詳細（テンプレート）
├── events.html ............. イベント情報
├── membership.html ......... 会員へのご案内
└── contact.html ............ お問い合わせ
```

### ページ間の導線

```
            ┌──────────┐
            │  トップ   │
            └────┬─────┘
  ┌─────┬─────┬──┼──┬──────┐
  ▼     ▼     ▼  ▼  ▼      ▼
同窓会 お知らせ イベント 会員案内 お問い合わせ
について  │
          ▼
```

- **お知らせ → 記事詳細**: 一覧から個別記事へ

## 各ページ詳細

### index.html — トップページ

トップページは各ページへの「ハブ」。最新情報のダイジェストを並べ、
各ページへの導線を作る。

#### Hero エリア
- **背景**: --bg-primary + 微細ノイズテクスチャ（SVGフィルター）
- **高さ**: 80vh
- **キャッチコピー**: Shippori Mincho / 大きめ
  - 「卒業しても、西高生。」
- **サブテキスト**: Noto Sans JP 300
  - 例: 「岡崎西高等学校 同窓会」
- **staggered reveal**: テキストが0.1s刻みで順次フェードイン（このページのみ）

```
（余白）

    卒業しても、西高生。

    岡崎西高等学校 同窓会

（余白）
```

#### お知らせダイジェスト
- **背景**: --bg-secondary
- **内容**: 最新のお知らせ（8/16同窓会開催、HP開設 等）
- **導線**: 「お知らせ一覧へ →」リンク（→ news.html）

#### イベントダイジェスト
- **背景**: --bg-primary
- **内容**: 2026年8月16日（日）同窓会開催予定
- **導線**: 「イベント情報へ →」リンク（→ events.html）

#### LINE登録バナー
- **背景**: --bg-secondary
- **内容**: 公式LINEへの登録を一言で案内 + ボタン
- **導線**: 「LINE登録・何回生か調べる →」リンク（→ membership.html）

### membership.html — LINE登録・会員案内

公式LINEへの登録を促すページ。
登録時に必要な「何回生か」をその場で調べられる計算機を設置する。

#### 何回生？計算機
- 生年月日を入力すると、自分が何回生かを自動計算して表示
- **計算ロジック**（JavaScript）:
  - 2026年度の卒業生 = 40回生（基準）
  - 高校卒業は18歳になる年度の3月
  - 4月1日生まれは前年度扱い（早生まれ対応）
  - 回生 = 生まれた年度 - 1968
    （例: 2008年度生まれ → 2008 - 1968 = 40回生）
- **入力**: 生年月日（年/月/日のプルダウンまたはdate input）
- **出力**: 「あなたは○○回生です」と大きく表示
- **デザイン**: シンプルなフォーム。結果表示は --navy 大文字で目立たせる

```
あなたは何回生？

生年月日を入力してください。
LINEの友だち追加時に回生の送信が必要です。

生年月日  [1990] 年 [4] 月 [15] 日

        ↓

  あなたは 22回生 です

```

#### 公式LINE登録案内
- LINE公式アカウントへの登録を案内
- QRコードまたは友だち追加ボタンを配置
- 登録手順を簡潔に説明（1. 友だち追加 → 2. 何回生かを送信）
- LINEの友だち追加ボタン: 緑（LINE公式色 #06C755）をそのまま使用

```
公式LINEに登録する

同窓会の最新情報はLINEでお届けしています。

  ① 下のボタンから友だち追加
  ② トーク画面で「○○回生」と送信

  [LINEで友だち追加]  ← LINE緑ボタン

  または QRコードから ↓
  [QRコード画像]
```

---

### contact.html — お問い合わせ

お問い合わせは公式LINEを第一窓口とし、LINE登録を促す構成にする。

#### 公式LINEでのお問い合わせ（メイン）
- LINEでのお問い合わせを推奨する旨を明記
- 友だち追加ボタン / QRコードを配置（membership.html と同じもの）
- 「まだ登録がお済みでない方は」→ LINE登録ページ（membership.html）へのリンク

#### メールでのお問い合わせ（サブ）
- LINEが使えない方向けにメールアドレスを掲載
- フォームは設置しない（シンプルにメールアドレスのみ）

```
お問い合わせ

お問い合わせは公式LINEからお願いします。

  [LINEで友だち追加]  ← LINE緑ボタン

  まだ登録がお済みでない方は →
  LINE登録・何回生か調べる

────────────────────────

LINEをお使いでない方は
下記メールアドレスへお問い合わせください。

  Mail: info@example.com
```
